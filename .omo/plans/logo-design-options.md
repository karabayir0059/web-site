# Logo Tasarım Konseptleri — "Web Site ve Otomasyon"

## Renk Paleti (sabit)
- Mor: `#6750A4` (brand) — güven, profesyonellik, teknoloji
- Altın: `#DAA654` (accent) — premium, sıcaklık, vurgu
- Siyah/beyaz fon kullanımı

---

## Konsept A — "The Link" (Geçmeli SO)

```
          ┌──────────┐
          │   ⭗ S   │  ← Altın "S", mor "O" halkası içinden geçiyor
          │    ⭘    │
          │         │  S = Site, O = Otomasyon
          │         │  İç içe geçmiş = bütünleşik hizmet
          └──────────┘
```

**Açıklama:**  
"S" harfi bir yol/akış gibi, "O" halkası ise onu çevreliyor.  
S, O'nun içinden geçiyor — site ve otomasyonun iç içe çalıştığını simgeliyor.  
Modern, teknoloji odaklı, akılda kalıcı.

**Animasyon:**
1. Önce S çizilir (soldan sağa)
2. Sonra O halkası belirir
3. İkisi birlikte glow yapar

---

## Konsept B — "The Node" (Düğüm)

```
          ┌──────────┐
          │   ⊚ S   │  ← Mor daire içinde altın S
          │   ╱ ╲   │  ← Bağlantı noktaları (web node)
          │  ●   ●  │
          │         │  Ağ/düğüm metaforu
          └──────────┘
```

**Açıklama:**  
Büyük bir mor daire (O), içinde geometrik bir S.  
Dairenin etrafında küçük bağlantı noktaları (web network).  
Kurumsal, sağlam, güven verir. Mimarlık/ofis havası.

**Animasyon:**
1. Daire belirir (genişleyerek)
2. Noktalar tek tek yanar
3. S ortada çizilir
4. Hepsi pulse eder

---

## Konsept C — "The Flow" (Akış)

```
          ┌──────────┐
          │   ∞~    │  ← Kesintisiz çizgi = S + O birleşimi
          │  ╭╮     │
          │  ╰╯     │  Gradyan mor → altın
          │         │  Akışkan, dinamik
          └──────────┘
```

**Açıklama:**  
Tek bir sürekli çizgi, önce S'yi sonra O'yu oluşturur.  
Gradyan mor → altın geçişi.  
Otomasyon akışını, kesintisiz hizmeti temsil eder.  
Sanatsal, premium, butik ajans havası.

**Animasyon:**
1. Çizgi baştan sona çizilir (stroke-dashoffset)
2. Gradyan rengi akar
3. Hafif dalgalanma efekti

---

## Konsept D — "Klasik SO" (Yan Yana)

```
          ┌──────────┐
          │  S  O    │  ← Özel tipografi (custom font)
          │  ██ ██   │  ← S: mor kare, O: altın daire
          │          │
          │          │  Temiz, okunabilir, klasik
          └──────────┘
```

**Açıklama:**  
Kalın bir S (mor dikdörtgen içinde) ve yanında O (altın daire).  
Basit, temiz, her yerde çalışır.  
En güvenli seçenek, kurumsal.

**Animasyon:**
1. Mor kare sol taraftan gelir
2. Altın daire sağ taraftan gelir
3. İkisi yan yana durur, hafif parıltı
